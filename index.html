Cara Menggunakan IG Tracker
Panduan lengkap step-by-step
1
Minta Data dari Instagram
Buka Instagram → Profil → ikon ☰ (tiga garis) → Pengaturan dan privasi → Pusat Akun → Informasi dan izin → Unduh informasi kamu. Pilih format JSON dan rentang tanggal Sepanjang waktu. Klik Buat file.
Instagram → Settings → Download Your Information → JSON format
2
Tunggu Email dari Instagram
Instagram akan mengirim email dengan link unduhan dalam beberapa menit hingga 48 jam. Klik link tersebut dan unduh file .zip ke perangkatmu.
3
Ekstrak File ZIP
Ekstrak (unzip) file yang sudah diunduh. Di dalam folder yang dihasilkan, navigasi ke subfolder connections/followers_and_following/. Di sana kamu akan menemukan file-file JSON berikut:
📄followers_1.jsonWajib
📄following.jsonWajib
📄recently_unfollowed_profiles.jsonOpsional
4
Upload File ke IG Tracker
Di halaman ini, klik masing-masing kotak upload dan pilih file JSON yang sesuai. Setelah kedua file wajib terupload, klik tombol "Mulai Analisis".
5
Nikmati Hasilnya!
Kamu akan mendapatkan laporan lengkap: siapa yang tidak follback, siapa yang kamu tidak follback, mutual followers, riwayat unfollow, dan insight statistik akun Instagram-mu. Semua data diproses 100% di browser-mu — tidak ada yang dikirim ke server manapun. 🔒
