Cara penggunaan
How to use IGTracker
Panduan ekspor data Instagram dan cara menggunakannya di sini.
Guide to exporting your Instagram data and using it here.
1
Minta ekspor data Instagram
Request Instagram data export
Buka Instagram → Pengaturan & Privasi → Pusat Akun → Informasi & Izin → Unduh Informasi Kamu. Pilih akun Instagram, pilih info yang diinginkan, dan pastikan format JSON.
Open Instagram → Settings & Privacy → Accounts Centre → Your information & permissions → Download your information. Select your account, choose what to include, and set format to JSON.
Settings → Download Your Information → JSON
2
Pilih hanya data yang kamu butuhkan
Select only what you need
Di halaman "Pilih info tertentu untuk diekspor", centang hanya kategori yang relevan: Pengikut dan mengikuti, Suka, Komentar, Pencarian terbaru, Login dan pembuatan profil, Topik Anda, Informasi iklan. Hilangkan centang Media untuk menghindari file besar.
On the "Select specific information to export" page, check only relevant categories: Followers and following, Likes, Comments, Recent searches, Login activity, Your topics, Ads information. Uncheck Media to avoid large files.
Jangan centang Media/Foto/Video — ukurannya bisa sangat besar dan tidak dibutuhkan di sini.
Do not check Media/Photos/Videos — these can be very large and are not needed here.
3
Tunggu email & download ZIP
Wait for email & download ZIP
Instagram mengirim email dengan link download dalam beberapa menit hingga 48 jam. Klik link dan download file .zip-nya.
Instagram emails you a download link within minutes to 48 hours. Click it and download the .zip file.
4
Ekstrak & temukan file JSON
Extract & locate JSON files
Ekstrak ZIP. Navigasi ke subfolder yang sesuai dan temukan file berikut:
Extract the ZIP. Navigate to the relevant subfolders:

5
Pilih fitur & upload di sini
Select features & upload here
Di Langkah 1, pilih hanya fitur yang kamu inginkan. Di Langkah 2, upload hanya file yang sesuai. Klik Analisis Data.
In Step 1, select only the features you want. In Step 2, upload the corresponding files. Click Analyze Data.
6
100% privat — data tidak keluar
100% private — data stays local
Semua pemrosesan dilakukan di browser menggunakan JavaScript. Tidak ada data yang dikirim ke server manapun. Kamu bahkan bisa offline sepenuhnya saat menggunakan alat ini.
All processing happens in your browser via JavaScript. No data is sent anywhere. You can even go fully offline before uploading — this tool works entirely locally.
