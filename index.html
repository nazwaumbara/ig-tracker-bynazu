Upload file JSON yang diperlukan
Upload the required JSON files
Catatan penting: Important note: Saat mengekspor data dari Instagram, pilih format JSON. Tidak perlu mendownload folder media (foto/video) — file JSON yang dibutuhkan berukuran sangat kecil (biasanya hanya beberapa KB). Cukup cari file JSON di dalam folder ZIP yang dikirimkan Instagram. When exporting data from Instagram, choose JSON format. You don't need to download any media folders (photos/videos) — the JSON files are very small (typically just a few KB). Simply locate the JSON files inside the ZIP Instagram sends you.
Cara penggunaan
How to use IGTracker
Panduan langkah-demi-langkah mengekspor dan menganalisis data Instagram kamu.
Step-by-step guide to exporting and analyzing your Instagram data.
1
Minta ekspor data dari Instagram
Request a data export from Instagram
Buka Instagram → Pengaturan & Privasi → Pusat Akun → Informasi & Izin → Unduh Informasi Kamu. Pilih akun Instagram kamu, lalu pilih format JSON.
Open Instagram → Settings & Privacy → Accounts Centre → Your information & permissions → Download your information. Select your Instagram account, then choose JSON format.
Settings → Download Your Information → JSON
2
Tentukan rentang tanggal
Set date range
Pilih rentang waktu sesuai kebutuhan. Untuk analisis relasi, pilih Sepanjang Waktu. Untuk riwayat aktivitas, bisa pilih 1 atau 3 bulan terakhir agar file lebih kecil.
Choose a time range based on your needs. For relationship analysis, select All Time. For activity history, 1–3 months keeps the file small.
3
Tunggu email dari Instagram
Wait for Instagram's email
Instagram akan mengirim email berisi link download dalam hitungan menit hingga 48 jam. Klik link tersebut dan download file .zip-nya.
Instagram will email you a download link within minutes to 48 hours. Click the link and download the .zip file.
4
Ekstrak ZIP dan temukan file JSON
Extract ZIP and locate JSON files
Ekstrak ZIP tersebut. Kamu hanya butuh file JSON — tidak perlu folder media. File-file yang diperlukan:
Extract the ZIP. You only need the JSON files — no need for media folders. Relevant files are located at:
connections/followers_and_following/followers_1.jsonWajibRequired
connections/followers_and_following/following.jsonWajibRequired
connections/followers_and_following/recently_unfollowed_profiles.jsonOptional

Jangan download folder yang berisi foto/video — ukurannya bisa sangat besar dan tidak dibutuhkan di sini.
Do not download folders containing photos/videos — they can be very large and are not needed here.
5
Pilih fitur dan upload
Select features and upload
Kembali ke bagian atas halaman ini. Pilih fitur yang kamu inginkan di Langkah 1, lalu upload hanya file JSON yang sesuai di Langkah 2. Klik Analisis Data.
Return to the top of this page. Select the features you want in Step 1, then upload only the relevant JSON files in Step 2. Click Analyze Data.
6
Data kamu tidak pernah keluar
Your data never leaves your device
Seluruh pemrosesan dilakukan di browser-mu menggunakan JavaScript. Tidak ada data yang dikirim ke server manapun. Kamu bahkan bisa memutus internet sebelum upload — alat ini tetap berfungsi penuh.
All processing happens in your browser using JavaScript. No data is sent to any server. You can even disconnect from the internet before uploading — this tool works entirely offline.
